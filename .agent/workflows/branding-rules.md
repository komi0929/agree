# ブランディングルール

## サービス名の表記

**正式名称: `agree`（全て小文字）**

### ルール

1. **サービス名は常に小文字の「agree」を使用する**
   - ❌ AGREE
   - ❌ Agree
   - ✅ agree

2. **適用箇所**
   - ページタイトル
   - メタデータ（title, description）
   - ヘッダー・フッター
   - 利用規約・プライバシーポリシー
   - コピーライト表記
   - alt属性
   - UIテキスト全般

3. **例外なし**
   - 法的文書（利用規約など）でも小文字を使用
   - 文頭でも大文字にしない

### 確認方法

新しいページやコンポーネントを追加する際は、以下のコマンドで大文字表記が残っていないか確認：

```bash
grep -r "AGREE\|Agree" --include="*.tsx" --include="*.ts" --include="*.json"
```
